TattooSIZER

Esta aplicación web permite a los usuarios cargar múltiples imágenes, redimensionarlas y organizarlas dentro de un lienzo de tamaño A4, optimizado a 300 DPI. La funcionalidad principal incluye la visualización previa del lienzo en una ventana emergente, asegurando que todo esté correcto antes de descargar la imagen final agrupada en formato PNG.

El proyecto fue desarrollado para un cliente que necesitaba una solución eficiente para combinar imágenes y prepararlas para impresión en formato profesional.

Barber Appointment Booking

Desarrollo de una página web interactiva para una barbería, diseñada específicamente para dispositivos móviles. La página permite a los usuarios seleccionar una fecha, elegir el tipo de servicio deseado y definir si es a domicilio o en el local. También incluye integración con Google Maps API para mostrar la zona de cobertura de los servicios a domicilio. Al confirmar los detalles, se genera un mensaje personalizado con la información seleccionada, que se envía directamente a WhatsApp para completar la reserva. El diseño es sencillo, funcional y enfocado en una experiencia de usuario fluida, sin necesidad de bases de datos ni frameworks adicionales.

Sassa - Financial App

Diseño y desarrollo de una página web estática en WordPress con Elementor para promocionar una app financiera tipo "Billetera Virtual". La página tiene un diseño moderno y profesional, enfocado en usuarios individuales. Incluye secciones informativas como 'Preguntas Frecuentes', 'Análisis y Reportes', y los planes de pago disponibles. El proyecto se realizó bajo un enfoque minimalista, priorizando una experiencia visual limpia, sin animaciones ni funciones backend, siguiendo las especificaciones del cliente.

Administración Clinica App

Aplicación de consola diseñada para administrar eficientemente las atenciones, médicos y pacientes de una clínica. El sistema incluye menús interactivos que permiten:

  • Agregar y buscar médicos y pacientes.
  • Ver la lista de atenciones.
  • Filtrar atenciones por médico y paciente.

El proyecto fue desarrollado en C++ utilizando estructuras de datos para garantizar un rendimiento óptimo. Pensado como una herramienta funcional para pequeñas clínicas, prioriza la usabilidad a través de una interfaz por texto clara y sencilla.
